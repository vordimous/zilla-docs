### format

> `enum` [ `binary`, `text` ] | Default: `text`

The format for the number.

### multiple

> `number`

Restrict the number to a multiple of a given number.

### range

> `string` | Pattern: `((?:\\(|\\[))(-?\\d+(?:\\.\\d+)?)?,(-?\\d+(?:\\.\\d+)?)?((?:\\)|\\]))`

An allowed range of values for the number.
